# 中国大陆海外跨境直连方案 (VLESS + REALITY)

## 一、 方案概述
本方案专为在复杂网络环境下（如学校、公司或受限运营商）需要稳定访问海外资源的客户设计。
*   **核心协议**：VLESS + REALITY（模拟标准 HTTPS 流量，抗封锁能力强）。
*   **传输方式**：TCP 传输（有效解决运营商对 UDP 的封锁/限速）。
*   **兼容性**：支持 Windows (PC)、Android、iOS 全平台。
*   **特殊适配**：针对服务器 443 端口被占用的情况，使用自定义 **8443** 端口。

---

## 二、 硬件准备：海外 VPS 购买
为了获得最佳的访问速度（低延迟、高带宽），建议选择**美国西部（洛杉矶、圣何塞）**或**日本、香港**的服务器。

1.  **推荐供应商**：
    *   **入门级**：DigitalOcean, Vultr, Linode（按时计费，适合测试）。
    *   **专业级（对华优化）**：BandwagonHost (搬瓦工 CN2 GIA 线路), DMIT, Akile。
2.  **系统要求**：**Ubuntu 22.04 LTS** (推荐) 或 Debian 11/12。
3.  **关键步骤**：在云服务商控制台的 **Security Group (安全组/防火墙)** 中，放行 **TCP 8443** 端口。

## 三、 服务端部署 (Ubuntu 22.04)

### 1. 开启 BBR 网络加速
BBR 是 Google 开发的拥塞控制算法，能显著提升跨太平洋远距离传输的速度。
```bash
echo "net.core.default_qdisc=fq" >> /etc/sysctl.conf
echo "net.ipv4.tcp_congestion_control=bbr" >> /etc/sysctl.conf
sysctl -p
```

### 2. 一键安装 Xray (VLESS + REALITY)
我们使用目前最稳定的一键脚本进行部署：
```bash
wget -P /root -N --no-check-certificate "https://raw.githubusercontent.com/mack-a/v2ray-agent/master/install.sh" && chmod 700 /root/install.sh && /root/install.sh
```
**安装向导说明：**
*   选择：**安装 / 更新**。
*   协议：选择 **VLESS + REALITY**。
*   端口：手动输入 **8443**。
*   SNI 目标域名（回落域名）：输入 `www.microsoft.com` 或 `www.yahoo.com`。
*   安装完成后，屏幕会输出一个 `vless://` 链接。**请完整复制并保存。**

## 四、 客户端配置指南

### 1. Windows PC 端 (v2rayN)
*   **软件下载**：从 GitHub 下载 `v2rayN-With-Core.zip`。
*   **导入配置**：复制 `vless://` 链接，在软件界面按 `Ctrl + V` 导入。
*   **核心设置 (VPN 模式)**：
    1.  底部 **“系统代理”**：选择“自动配置系统代理”。
    2.  底部 **“路由”**：选择“绕过大陆 (Bypass mainland)”。
    3.  **开启 TUN 模式**：点击“设置”->“参数设置”->“v2rayN设置”->“开启TUN模式”设为 `True`（需管理员权限）。
    *   *好处：接管整台电脑流量，支持游戏和命令行开发工具。*

### 2. Android 端 (v2rayNG)
*   **软件下载**：Google Play 商店或 GitHub 下载 v2rayNG。
*   **导入配置**：点击右上角 `+` 号，选择“从剪贴板导入”。
*   **分流设置**：
    1.  点击左侧菜单 -> 设置。
    2.  **分路由模式**：选择“绕过局域网及大陆地址”。
    3.  **分应用代理**：仅勾选需要加速的应用（如 Chrome, Telegram, YouTube）。
*   **连接**：点击右下角蓝色“V”图标启动。

### 3. iOS 端 (V2Box - 免费方案)
*   **软件下载**：前往 App Store 搜索安装 **V2Box - V2Ray Client**。
*   **导入配置**：
    1.  点击底部 **Configs** 页面。
    2.  点击右上角 `+` 号 -> **Import from Clipboard**。
*   **连接与分流**：
    1.  回到 **Home** 页面，点击 **Tap to Connect**。
    2.  进入 **Settings** -> **Routing Mode** -> 选择 **Bypass Mainland China**。
    *   *注：iOS 建议保持手机系统时间为“自动设置”，否则会导致连接失败。*

## 五、 方案优势与常见问题

### 1. 为什么选择 8443 端口？
由于您的 443 端口已被现有业务占用，8443 是最常用的 HTTPS 备选端口。REALITY 协议在 8443 上依然保持着极高的伪装度，不容易被防火墙识别。

### 2. 为什么需要开启 TUN 模式？
TUN 模式会在系统中创建一个虚拟网卡，将普通的代理升级为“系统级 VPN”。这对于没有内置代理设置的软件（如 Zoom 会议、SSH 远程连接、外服游戏）是必须的。

### 3. 注意事项
*   **时间同步**：VLESS 协议对时间非常敏感，请确保客户端和服务器的时间误差不超过 90 秒。
*   **MTU 优化**：如果在部分 WiFi 下网页加载缓慢，请在客户端设置中将 MTU 修改为 `1280`。
*   **安全提示**：请勿将生成的 `vless://` 链接公开，以防服务器 IP 被恶意举报。
