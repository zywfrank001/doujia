# 100个英语常用词根

文章知识来自李平武老师《英语词根和单词的说文解字》，自己在读的时候希望将知识汇总到互联网上，方便大家学习和使用。

## 词根由来与派生规则

英语词根来源主要有三个部分
{% markmap %}
# 词根来源
## 古英语  
## 外来语
### 拉丁语
### 希腊语
{% endmarkmap %}
这些词根构成了90% 的英语单词，而且构词逻辑有迹可寻，遵循规律

我们将词根分为两种
{% markmap %}
# 词根分类
## 自由词根
### 可独立组词
## 粘连词根
### 不可独立组词
{% endmarkmap %}
自由词根基本来自古英语中，粘连词根来自外来语中

综上所述，英语词根的来源和分类是英语词汇学习的重要基础，有了这些词根就可以构词了
基本上构词即词根和词缀合成一个新的词
构词基本上可以分为一下三类

{% markmap 500px %}
# 构词分类
## 零派生无词缀
### 自由词根
### 粘连词根+自由词根
#### manu+script=manuscript 手稿
### 自由词根+自由词根
#### 比如 black+board=blackboard 黑板
#### 比如 honey+moon=honeymoon 蜜月
## 有词缀
### 一级派生
#### 前缀+粘连词根
#### 粘连词根+后缀
#### 前缀+粘连词根+后缀
#### 前缀+粘连词根+后缀+后缀
### 二级派生
#### 前缀+自由词根
#### 自由词根+后缀
#### 前缀+自由词根+后缀
#### 前缀+自由词根+后缀+后缀
{% endmarkmap %}

## 本章总结
总的来说，了解了英语单词的构词法，可以知道大多数的单词都是由其他单词或词根派生来
一般来说···词根···，···连词符号··· ，···前缀···，···后缀··· 就是觉得多数单词的构词单元。
